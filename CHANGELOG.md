# Changelog
Все заметные изменения в этом проекте будут документироваться в этом файле.

## [1.2.0] - 2026-03-30

### Added
- Добавлена возможность авторизации в приложении через Apple ID.
- В настройки интерфейса добавлен переключатель на темную тему.

### Changed
- Обновлен дизайн карточек задач: теперь они отображаются с современными закругленными углами.

### Deprecated
- Старый метод экспорта данных в формате XML признан устаревшим и будет полностью удален в версии 2.0.0.

### Removed
- Удален модуль интеграции с Twitter в связи с прекращением поддержки данной функции.

### Fixed
- Исправлена критическая ошибка, из-за которой приложение закрывалось при вводе специальных символов в поле электронной почты.

### Security
- Обновлена библиотека шифрования для устранения уязвимости безопасности (CVE-2026-0199).
