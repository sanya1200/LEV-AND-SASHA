# Changelog

Все заметные изменения в этом проекте будут фиксироваться в данном файле.

## [1.2.0] — 2026-03-30

### Добавлено
- **Авторизация через Apple ID**: Теперь пользователи могут быстро и безопасно входить в систему.
- **Новая система тегов**: Добавлена возможность цветовой маркировки задач.
- **Локализация**: Добавлена поддержка русского и казахского языков в интерфейсе.

### Исправлено
- Исправлена ошибка, из-за которой приложение аварийно завершало работу при попытке очистить пустой список задач.
- Оптимизировано потребление оперативной памяти при фоновой работе.

### Изменено
- Обновлен дизайн главного экрана для улучшения читаемости на мобильных устройствах.
- Минимальная версия ОС теперь — Android 10+ / iOS 15+.
